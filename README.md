# Resume Source

This repository hosts the source code for a live-hosted professional resume, synthesized through AI-assisted data aggregation and custom design iterations. The project is deployed via the DigitalOcean App Platform, utilizing a continuous deployment pipeline that automatically pushes repository updates to the live production environment. It represents a streamlined, modern approach to maintaining a high-availability professional profile with minimal manual overhead.

**Live Deployment:** [https://resume.raykooyenga.com](https://resume.raykooyenga.com)
